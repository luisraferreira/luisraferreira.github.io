Métodos dos arrays

a.pop() -> elimina o último elemento do array

a.shift() -> elimina o primeiro elemento do array

a.push() -> põe um elemento novo no fim do array

O caso especifico do splice

O splice serve não só para cortarmos elementos, mas também para pormos elementos onde quisermos.

a.splice(index, deleteCount, item1, item2,..., itemX)
